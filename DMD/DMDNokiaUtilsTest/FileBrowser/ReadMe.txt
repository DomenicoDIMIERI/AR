Required software environment:
------------------------------
- Microsoft Windows 2000 or XP
- Microsoft Visual Studio 2005
- Nokia PC Suite 6.8 or later (www.nokia.com/pcsuite)

Compiling instructions:
-----------------------
- Open file VBFileBrowser.NET.sln to Visual Studio 2005 and build

Module descriptions:
-------------------
FileBrowser.vb:
Main form of VBFileBrowser.NET example application. 

FileListBox.vb:
Implementation of file browser's file list box.

PhoneListBox.vb:
Implementation of file browser's phone list box.

FileOperationListener.vb:
Class for showing the progress status of file operations to user.  

FRM_Rename:
Form for typing new name when renaming phone file or folder.

FRM_Create:
From for typing new name when creating new phone folder.

Common.vb:
Main module of VBFileBrowser.NET example application.

PCCAPIUtils.vb:
Helper utilities, error management.

CONA*.vb, PCC*.vb:
Local Connectivity API definitions, converted from C++ Header files.
